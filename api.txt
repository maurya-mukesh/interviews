ROLE: ['customer', 'admin'] => have access to below API

1.Auth: (login needed for user-profile)
  POST:  http://localhost:5000/api/auth/user/register
  POST:  http://localhost:5000/api/auth/user/login 
  GET :  http://localhost:5000/api/auth/user-profile/:id

2.Bookings : Login needed
  POST:  http://localhost:5000/api/bookings
  GET :  http://localhost:5000/api/customers/:customerId/bookings
  GET :  http://localhost:5000/api/vehicles/:vehicleId/bookings

3.Customer API: Login needed (only for admin)
  POST: http://localhost:5000/api/customers
  PUT : http://localhost:5000/api/customers/:id
  DEL : http://localhost:5000/api/customers/:id

4.Statistics API : Public API 
  GET : http://localhost:5000/api/statistics/most-popular-vehicle
  GET : http://localhost:5000/api/statistics/bookings-per-vehicle
  GET : http://localhost:5000/api/statistics/average-rental-duration

5.Vehicles API : (Only For Admin: Login Needed)
  POST: http://localhost:5000/api/vehicles/new-vehicle
  PUT : http://localhost:5000/api/vehicles/update/:id
  DEL : http://localhost:5000/api/vehicles/delete/:id
  GET : http://localhost:5000/api/vehicles
  GET : http://localhost:5000/api/vehicles/maintenance/:vehicleId
